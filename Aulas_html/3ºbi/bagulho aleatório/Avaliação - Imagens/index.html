3- Responsive Images

As imagens responsivas se ajustarão automaticamente ao tamanho da tela. Se você quiser que uma imagem diminua se for necessário, mas nunca aumente para ser maior que seu tamanho original, adicione o seguinte: "max-width: 100%; height: auto;"

Redimensione a janela do navegador para ver o efeito:

4- Center an Image

Para centralizar uma imagem, defina as margens esquerda e direita autoe transforme-a em um blockelemento.

Use: "display: block; margin-left: auto; margin-right: auto; width: 50%;"
